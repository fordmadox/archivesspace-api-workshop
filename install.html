Installation notes

These are slides authored in Markdown. The website is produce with standard Unix/Linux utilities plus three open source utilities I wrote–

  • findfile locates the files needed to build site and presentation
  • md2slides converts a single markdown file into a series of HTML pages that serve as slides.
  • shorthand converts the markdown applying templating for the website
