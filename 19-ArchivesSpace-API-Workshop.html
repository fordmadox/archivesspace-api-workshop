3. Authentication

Send our username and password

We need pass our username and password in our request. We need to keep track of the response. You don’t need to type of the python comments.

Close the text editor and go back in the shell. Type the following (you can skip the lines starting with ‘#’)

    username = input('username (e.g. admin): ')
    # We want to use getpass.getpass('password: ') so the
    # password doesn't get echoed to the screen. In IDLE it'll show read and
    # echo the password, ignore this for testing and development.
    password = getpass.getpass('password: ')
